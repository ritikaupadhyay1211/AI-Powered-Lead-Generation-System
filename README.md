# AI-Powered Lead Generation System

---

## Project Overview

The **AI-Powered Lead Generation System** is an enterprise-grade Lead Management solution built in a Salesforce Developer Edition environment[span_0](start_span)[span_0](end_span). This project focuses on streamlining inbound pipeline velocity by replacing manual, error-prone data collection with automated validation, role-based access controls, and cutting-edge generative AI capabilities[span_1](start_span)[span_1](end_span).

By merging traditional declarative development (validation rules, custom schemas, and automated flows) with the **Agentforce Prompt Builder** tool, this solution maintains strict database hygiene while letting sales agents generate highly descriptive lead summaries with a single click[span_2](start_span)[span_2](end_span).

---

## System Configurations

* **Lead Score (`Lead_Score__c`):** Number field to record priority[span_3](start_span)[span_3](end_span).
* **Lead Category (`Lead_Category__c`):** Custom picklist mapping Hot, Warm, and Cold states[span_4](start_span)[span_4](end_span).
* **Product Interest (`Product_Interest__c`):** Picklist covering Training, Consulting, and Certification[span_5](start_span)[span_5](end_span).
* **AI Summary (`AI_Summary__c`):** Long Text Area capturing cognitive summaries[span_6](start_span)[span_6](end_span).
